# 🖼️ ภาพประกอบการใช้งาน

## เมนูหลัก

เมื่อเปิด Google Sheets จะเห็นเมนู **🏋️ WTH Fitness** ด้านบนเมนูบาร์

```
File  Edit  View  Insert  Format  Data  Tools  Extensions  Help  [🏋️ WTH Fitness]
                                                                        ↓
                                        ┌──────────────────────────────┐
                                        │ ⚙️ ตั้งค่าระบบ              ▶│
                                        │ ──────────────────────────  │
                                        │ 📊 จัดการข้อมูล             ▶│
                                        │ ──────────────────────────  │
                                        │ 🔐 API Settings            ▶│
                                        │ ──────────────────────────  │
                                        │ ❓ คู่มือการใช้งาน           │
                                        └──────────────────────────────┘
```

## 💡 เคล็ดลับการใช้งาน

### ดูสถิติก่อนเสมอ
```
🏋️ WTH Fitness → 📊 จัดการข้อมูล → 📈 ดูสถิติการใช้พื้นที่
```

### Archive เป็นประจำ
```
ทุก 6 เดือน:
🏋️ WTH Fitness → 📊 จัดการข้อมูล → 📦 Archive ข้อมูลเก่า
→ เลือกวันที่: 6 เดือนก่อน
→ เลือก: TestResults ✅ BodyMeasurements ✅
```

### ลบข้อมูลซ้ำหลัง Import
```
หลัง Import ข้อมูล:
🏋️ WTH Fitness → 📊 จัดการข้อมูล → 🧹 ลบข้อมูลซ้ำ
→ เลือก Sheet ที่ Import
```

### ลบข้อมูลเก่าแบบปลอดภัย
```
🏋️ WTH Fitness → 📊 จัดการข้อมูล → 🗑️ ลบข้อมูลเก่า
→ เลือกวันที่: 1 ปีก่อน
→ เก็บล่าสุด: 10 รายการ/คน ← สำคัญ!
```
